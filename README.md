# The Beatles App

 Jogo de perguntas a respeito da banda Os Bealtes, desenvolvindo como assessment para a disciplina Fundamentos de Android do instituto Infnet.

**Tecnologias empregadas**

- Constraint Layout
- DataBinding
- Navigation
- SafeArgs
- Lifecycle
- Fragments
- ViewModel
- LiveData 
- Share
- Drawermenu
- ReciclerView
